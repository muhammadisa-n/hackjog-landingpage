# Usage

### Clone Repo
`git clone https://github.com/muhammadisa226/crime-detection-api`

### Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Compile and Minify for Production

```sh
npm run build
```
